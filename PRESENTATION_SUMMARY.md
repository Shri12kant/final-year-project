# CommunityHub - Quick Presentation Summary

## Slide 1: Project Introduction

**CommunityHub** - A Modern Social Community Platform

- Similar to Reddit
- Users can create communities, share posts, vote, and interact
- Full-stack application with modern technologies

---

## Slide 2: Tech Stack

### Frontend
- React 18 + TypeScript
- Tailwind CSS (Styling)
- Vite (Build tool)
- React Query (Data fetching)
- Zustand (State management)

### Backend
- Spring Boot 3.x + Java 17
- Spring Security + JWT
- MySQL Database
- JPA/Hibernate (ORM)

### Deployment
- Frontend: Vercel
- Backend: Railway
- GitHub: Version Control

---

## Slide 3: Key Features

1. **User Authentication**
   - Register/Login with JWT
   - Email verification
   - Secure password handling

2. **Communities**
   - Create custom communities
   - Join communities
   - Search communities

3. **Posts**
   - Text and media posts
   - Image/Video upload (50MB max)
   - Sort by Hot/New/Top
   - Search posts

4. **Interactions**
   - Upvote/Downvote
   - Comments
   - User profiles
